TAList 

Companion object TAList

sealed abstract class TAList[F[_, _], A, B] extends Serializable

A type-aligned cons list.

In a type-aligned cons list, the "output" (or right) type of a node is equivalent to the "input" (or left) type of the node after it. For example, you could chain functions in a type aligned list like String => Option[Int], Option[Int] => Long, Long => Boolean.
