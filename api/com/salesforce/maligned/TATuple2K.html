TATuple2K 

Companion object TATuple2K

abstract class TATuple2K[F[_], G[_]] extends AnyRef

Equivalent to cats.data.Tuple2K but uses an existential type parameter for the pivot (A) type. This allows instances that share the same F and G types to be considered the same type, even if they don't share the same pivot type. This feature is being used to enable tail-recursive traversal across type-aligned structures such as in TANonEmptyList.reduceLeft and TANonEmptyList.reduceRight.

Inspired by Tomas Mikula's APair.
