  1.  trait TAListGenerator[F[_, _], S[_], L, R] extends AnyRef

    Supports generating pseudorandom TAList structures.

    Supports generating pseudorandom TAList structures.

    If you create a TAListGenerator instance for a type, you can then use the methods in TAListGen to create org.scalacheck.Gen instances for type-aligned lists of that type.

    F

    the effect type of the type-aligned list (for example Function1)

    S

    a type encapsulating any state required to generate the preceding element in the list

    L

    the input type of the generated list

    R

    the output type of the generated list
